# 💳 MaisCtrl

Sistema web de controle financeiro pessoal.

## 💡 Problema
Muitas pessoas perdem o controle de:
- assinaturas mensais
- parcelas de cartão
- gastos recorrentes
- valores a receber

## 🎯 Solução
O MaisCtrl centraliza todas essas informações em um único sistema,
com foco em simplicidade e organização visual.

## 🚀 Funcionalidades
- Controle de assinaturas
- Parcelas mensais
- Devedores
- Organização de gastos
- (em desenvolvimento) Fatura de cartão por banco

## 🧱 Como foi desenvolvido
- Plataforma: **Lovable**
- Estrutura do sistema, regras e lógica definidas por mim
- Foco em usabilidade e clareza das informações

## 🛠 Tecnologias envolvidas
- Front-end: HTML, CSS, JavaScript
- Back-end: lógica de dados e regras de negócio
- Plataforma: Lovable
